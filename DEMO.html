üîß How to Run the Full Application

  1. Start Docker Desktop: Search for "Docker Desktop" in Windows Start Menu and launch it
  2. Wait for Docker to be ready: Look for the whale icon in your system tray (bottom-right)
  3. Open Terminal in project folder: C:\Users\ASUS\LEARNING_PROJECT
  4. Run: docker-compose up -d
  5. Access services:
    • Frontend: http://localhost:3000
    • Course API: http://localhost:3001
    • User API: http://localhost:3002
    • Notification API: http://localhost:3003
